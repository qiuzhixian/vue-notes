### vue 目录结构(2.0 版本)

```
├── node_modules     # 依赖包目录
├── public
│   ├── favicon.ico  # ico图标
│   └── index.html   # 首页
├── src
│   ├── assets       # 样式图片目录
│   ├── components   # 组件目录
│   ├── views        # 页面目录
│   ├── App.vue      # 父组件
│   ├── main.js      # 入口文件
│   ├── router.js    # 路由配置文件
│   └── store.js     # vuex状态管理文件
├── .gitignore       # git忽略文件
├── babel.config.js  # babel配置文件
├── package.json     # 包管理文件
└── package-lock.json # 记录 package的具体来源和版本号
```
